# Smart Notes Project üìù

This is a Node.js + Express + MongoDB project that allows users to:
- Register/Login
- Create, edit, and delete notes
- Use AI to summarize notes

## Setup

1. Clone the repo
2. Run `npm install`
3. Add your `.env` file
4. Run `npm run dev`

## API Endpoints

- `POST /signup`
- `POST /login`
- `POST /logout`
- `PATCH /forgetPassword`
- `PATCH /resetPassword`
- `POST /note/summarize`

## Technologies Used

- Node.js
- Express
- MongoDB
- Graphql
- OpenAI API
